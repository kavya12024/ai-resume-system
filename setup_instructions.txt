## ğŸš€ Quick Start

### Prerequisites
- Java 11 or higher
- Python 3.7 or higher
- MySQL (XAMPP recommended)
- Git (optional)

### Step 1: Database Setup

1. **Start XAMPP**
   - Launch XAMPP Control Panel
   - Start Apache and MySQL

2. **Create Database**
   - Open phpMyAdmin: http://localhost/phpmyadmin
   - Create new database or import `database/create_tables.sql`
   - Copy the SQL content and paste into phpMyAdmin
   - Execute the query

3. **Verify Connection**
   - Database: `interview_system`
   - User: `root`
   - Password: (empty by default)

### Step 2: Python AI Microservice Setup

1. **Install Python Dependencies**
   ```bash
   cd python_ai
   pip install flask
   pip install requests
   ```

2. **Run Python Service**
   ```bash
   python app.py
   ```
   - Service will start on: http://localhost:5000
   - Test health: http://localhost:5000/health

### Step 3: Java Backend Setup

1. **Compile Java Code**
   ```bash
   cd backend_java/src
   javac MainServer.java DBConnection.java
   ```

2. **Add MySQL JDBC Driver**
   - Download: `mysql-connector-java-8.0.x.jar`
   - Place in `backend_java/src` folder
   - Compile with driver:
   ```bash
   javac -cp mysql-connector-java-8.0.x.jar MainServer.java DBConnection.java
   ```

3. **Run Java Server**
   ```bash
   java -cp .:mysql-connector-java-8.0.x.jar MainServer
   ```
   - Backend will start on: http://localhost:8080

### Step 4: Frontend

1. **Open Browser**
   - Navigate to: http://localhost:8080
   - Frontend is served by Java server

2. **Create Account or Login**
   - Test credentials: john@example.com / password123
   - Or sign up for new account

## ğŸ“‹ API Endpoints

### Authentication
- `POST /api/login` - User login
- `POST /api/signup` - User registration

### Resume & Skills
- `POST /api/uploadResume` - Upload resume and extract skills
- `GET /api/getSkills?userId=1` - Get user skills

### Interview
- `POST /api/startInterview` - Start new interview session
- `POST /api/nextQuestion` - Get next question
- `POST /api/evaluateAnswer` - Evaluate answer and get score

### Progress
- `POST /api/saveProgress` - Save interview progress
- `GET /api/history?userId=1` - Get interview history

## ğŸ› ï¸ Troubleshooting

### Java Server Issues
- **Port 8080 already in use**: Change PORT in MainServer.java
- **MySQL connection failed**: Check XAMPP MySQL is running
- **Driver not found**: Ensure MySQL JDBC driver is in classpath

### Python Service Issues
- **Port 5000 already in use**: Change port in app.py
- **Module not found**: Run `pip install -r requirements.txt`

### Frontend Issues
- **Blank page**: Check browser console for errors
- **API calls fail**: Verify Java server is running on http://localhost:8080
